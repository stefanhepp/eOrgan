---- Specification ----
- 3 manuals (61 keys), pedalboard (29 keys)
- 12 toe studs, 3 pedals (choir/swell, cresc.)
- 64 drawknobs, 3 banks
- Tremolo/Vibrato for II,III,P
- 57 pistons

---- Interface ----

- 29 Pedale, C - e
- Switch: On/Off/Setup
- Rear:
  - Midi-Out-Buchse
  - VCC 5V (Buchse)

- Front, Pedale
  - 3 Pedale: Links: Chor Schwellerpedal, Mitte: Schwellwerk Schwellerpedal, Rechts: Crescendo Pedal
  - 16 toe studs: 4 Hauptsetzer (Links), 4 Pedal-Setzer (Rechts), 
    II/P, I/P, Seq. Prev/Next (rechts neben Schweller), Turn page, 3 programmable pistons (links neben Schweller)
- Front, Weitere
  - Beleuchtung Pedal (5 Weiß)
  - 2xRiegel Frontklappe (oben)
  - 2xBoxen-Verkleidung (über Setzer)
- Rear, PC
  - VCC 220V
  - USB, LAN, VGA, PS2
  - SPDIF, Box, LineOut, Mic, Midi-In/-Out/Gameport
- Rear, Weitere
  - VCC 220V
  - Riegel Konsole-, Gehäusedeckel
  ? Amp In (loopback Soundcard?)
  ? Amp Out/Boxen 2.1/5.1

  - 3 Manuale a 61 Tasten C - C (Kanaleinstellung intern on controller)
  - 50 pistons
- Front, Main (links)
  - Main Key Lock (Pwr Main/Pwr PC)
  - Pwr Main On/Off (+ Main Pwr LED)
  - Pwr PC (Taster, + PC Pwr LED, +HDD Led)
  - Pwr Audio On/Off (+ Audio LED)

  - 2x USB 2.0
- Rear
  - VCC (Standalone only)
  - Midi Out
  - Midi In (to Manual)

o) Setzer, Koppler Spieltisch
- Effects (links), Taster
  - Tremolo, Vibrato/Chorus (+LED, per I,II,III)
  - Speed Slow/Fast (Taster +LED, global, individual settings only per comb.setup; slow+fast disabled: normal speed (?))
  - Slow Transition (Taster, +LED, global, affects transition for tremolo/vibrato and leslie)
  - Leslie On (+Led, enables additional leslie vibrato effect, globally (affected divisions configured per menu))
  - Leslie Fast (+Led, switch to fast vibrato, transition speed configured per menu, depends on Slow transistion switch)
  - Menu (Effects Menu)
  - Setzer: 1, 2, 3, 4, 5, 6, C, R (affect only Effects)
  - Amount (Poti, global, Settings per Division/Effect über LCD)
- Coupler (links), Taster mit LED
  - Generell:
    - CR: Crescendo, Crescendo Pedal Registrierung zu Registern dazu setzen
    - Off: Unison Off, Manual spielt nicht auf eigener Division 8' (betrifft nicht 16'/4' Koppler, CR)
    - 4': Oktav-Koppel, 16': Suboktav-Koppel
    - Split pedal: Split pedal, default coupler control upper half, lower half is P'
  - Schwellwerk (9): II/III, I/III, P/III, 4' II/III, 4' I/III, 16', 4', Off, CR/III
  - Hauptwerk (11):  III/II, I/II, P/II, 16' III/II, 16' I/II, 4' III/II, 4' I/II, 16', 4', Off, CR/II
  - Chor (11):	     III/I,  II/I, P/I,  4' P/I,  16' II/I,  16' III/I, 4' III/I,  4' II/I, 16', 4', Off, CR/I
  - Pedal (13):	     I/P, II/P, III/P, 4' I/P, 4' II/P, 4' III/P, 4',  Off, CR/P, II/P', I/P', Off P', Split P'
  - Setzer: 1, 2, 3, 4, 5, 6, C, R (affect only Coupler)
- Setzer (rechts), Taster (ohne LEDs)
  - Generell
    - K: Koppler + Effekte
    - Global: Alle Divisions + K
    - Effekte bei Stops dabei, Effekte immer auf alle Bänke einer Division
    - Alle aktiven Stops aller Bänke werden gespielt, unabhängig von Bank-Select
  - 8 Setzer global, übernimmt alle Divisions, + Koppler falls SetK, ändert/setzt nur aktive Bank
  - 8 programmable pistons 
  - 8 Presets: Lädt Global/Div. Setzer, ..
  - Reset (Global,I,II,III,P,K), aktuelle Stops verwenden
  - Clear (Global,I,II,III,P,K), alle Stops/Koppler deaktivieren
  - Bank Select (B1,B2,B3, + LED): Stops affect Bank, aktuelle Bank für Set/Setzer/Koppler/R/C/CR
  - Bank/Set (S/B, global, + LED): Set übernimmt aktuelle/alle Bänke; Setzer/Clear/Reset ändert nur aktuelle/alle Bänke
  - Bank/Koppler (K/B, global, + LED): Koppler ändert nur Koppel auf aktueller Bank/allen Bänken
  - Set Koppler (K/S, global, + LED): Set übernimmt auch Koppler; Setzer ändert Koppler, falls gesetzt
  - Couple Global (G/S, global, + LED): Set stores registration from all stops on all/active banks to divisional pistons, activates stops for divison
    only (without division coupler)
  - Clear Global (Global, + LED): Clear all global single register links (LED shows if any are active)
  - Set (global, + LED), Druck auf Setzer übernimmt aktuelle Registrierung
  - Hold (global, + LED), Registrierung erst ändern, wenn Hold deaktiviert
  - Add (global, + LED), bei Set: gespeicherter Setzer addiert immer; Sonst: Setzer addiert wenn Add aktiv
  - Clear (Global, + LED), wie Add, deaktiviert alle gesetzten Register
  - Menu, ruft Setzermenü auf (Save,Load,Transpose,Crescendo,Effects,Sequencer..)
  - Sequencer next, prev
  - Page next, prev
- Taster (unter Manualen), Taster rund
  - Generell:
    - PP,P,F,FF mit LED, ändert nicht Registrierung/Setzer, überschreibt Setzer/Stops
    - Alle anderen ohne LED, setzen Status in Panel
    - Setzer: gleicher Setzer wie in Panel, optional (per Menü) unabhängig für mehr Kombinationen
    - 1-4: Setzerplätze, C=Clear, R=Reset
  - Schwellwerk (22): Progam: 1, 2, ,3 ,4, 5, 6, 7, 8; Page Turn prev/next; Schwell: I/III, II/III PP, P, F, FF, 1, 2, 3, 4, C, R
  - Hauptwerk (22):   Global: PP, P, F, FF, 1, 2, 3, 4, C, R; Seq  prev/next; Great: PP, P, F, FF, 1, 2, 3, 4, C, R
  - Chor (24):	      Pedal: II/P, PP, P, F, FF, 1, 2, 3, 4, C, R; Chor: III/I, II/I, P/I PP, P, F, FF, 1, 2, 3, 4, C, R

  - Great Preset 1..6

  - Menu
  - Leslie on,fast,effect on hammond only
- Setzer:
  - Select stop from all divisions for divion (coupler per stop)

o) only per menu

- Second Touch: Add Effects, Add Coupler using Second-Touch/Aftertouch
- Percussion Reiterate: per Menu
