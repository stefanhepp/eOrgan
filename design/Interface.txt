- Hold: if enabled, only change registration when hold is disabled
- Set Global (Borrow): if enabled, set division combination will set all stops on all divisions
- Store Couplers: if enabled, store division/global couplers instead / in addition to stops (2x press, blinks ?)

===================

    - Combinations 1-8
- Page Turn <, >
- Swell
    - Solo/III, II/III, I/III
    - 16'/4': Toggle 16'/4' coupler, Hold + press coupler to enable 16'/4' coupler from division (?)
    - Off: 1x Press: Off Toggle, 2x Press: Tutti
    - Combinations 1-5, Clear (C), Crescendo Enable (CR)

    - Solo/P, III/P, II/P, I/P

Console Toe Pistons
===================
o) Left side
- Top row: 4 programmable: Set, Hold, Clear (global), Page Turn back
- Bottom row: 4 Global combinations, Sequencer back

o) Center (swell/expression shoes)
- Swell III (Swell)
- Swell I (Choir)
- Crescendo

o) Right side
- Top Row: II/P, I/P, Page Turn forward
- Bottom row: Sequencer forward, 4 Pedal combinations

o) Connectors
- Mini Din (Data In): 12V DC In, I2C
- Mini Din (Data Out): 12V DC In, I2C -> Pedal Board
- MIDI Out
- Configure Button
- 10pin flatcable left pistons
- 10pin flatcable right pistons
- 5pin JST Swell shoes

o) Configuration
- Set Midi Channel 1-16
- Reset Calibrate Swell shoes
